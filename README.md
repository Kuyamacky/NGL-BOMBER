# NGL BOMBER

A simple script for sending messages via NGL.

---

## Features

| Feature                     | Status |
|-----------------------------|--------|
| Troll ngl messages          | ✅     |
| Easy to use                 | ✅     |
| Works on Termux             | ✅     |
| Auto Update                 | ✅     |

---

## Installation

```bash
# Install Git
pkg install git
```

# Clone this repository
```bash
git clone https://github.com/Kuyamacky/NGL-BOMBER.git
```

# Go to the cloned directory and run the script
```bash
cd NGL-BOMBER && bash ngl.sh
```
